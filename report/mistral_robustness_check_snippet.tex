\paragraph{Independent local-judge robustness check.}
As a robustness check, we rescored a balanced 180-response subset using a local Mistral judge through Ollama, which is a different model family from the Groq-hosted Llama model used in the main pipeline. The subset contained 60 responses per condition and was balanced across perceived race groups (20 White, 20 Black, and 20 Asian per condition). The same broad fairness pattern held: the Baseline condition had a larger maximum racial quality gap (0.150) than Persona-Blind (0.100), while Reranked remained intermediate/similar to Baseline in this local-judge subset (0.150). Because this check uses a local model with coarse 4/5-heavy scores, we treat it as supporting evidence rather than as a replacement for the main full-dataset audit.
